   What is '객체(Object)'??
   클래스와 인스턴스는 맴버(변수, 메소드)를 가질 수 있음.
   클래스 맴버: 키워드- static
   

  • 클래스 변수: 클래스에 따라 만들어진, 모든 인스턴스들이 갖고 있는 변수(인스턴스에도 영향을 미침)
    •    
  • 클래스 메소드: 인스턴스 맴버에 접근x


    •    인스턴스 맴버
       
  • 인스턴스 변수: 인스턴스마다 다르게 갖는 변수(클래스 자체에 영향을 미치지 않음)
    •    
  • 인스턴스 메소드: 클래스 맴버에 접근o


    •    ex)
       class C1{ //클래스
        static int static_variable = 1; //클래스변수
        int instance_variable = 2; //인스턴스변수
        static void static_static(){ //클래스메소드
        System.out.println(static_variable);
        }
        void instance_static(){ //인스턴스메소드
        System.out.println(static_variable);
        }
        void instance_instance(){
        System.out.println(instance_variable);
        }
       }
