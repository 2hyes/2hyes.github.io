1. 스타트업 '모빌리오' SW인턴

2019.10.21 ~ 2020.02.06
2020.10.22 ~ 재직중
성남시 소재의 스타트업 '모빌리오'에서 SW인턴으로 근무

활용 기술

language - C#, DBMS - MySQL, MariaDB, etc - Grafana

업무 내용

  1. 센싱 데이터 송수신 및 모니터링 프로그램 디버깅 및 안정화
  2. 프로그램 배포 및 환경 구축
  3. 센서 네트워크망 형성(센서(Ubuntu기반) → 라우터 → PC → AWS 클라우드서버 DB)
  4. Database 관리
  5. 오픈소스 대시보드 프로그램인 grafana를 활용하여, 실시간 데이터를 시각화하는 대시보드 구현
네트워크를 통한 데이터 송수신, AWS 서버 및 사내 윈도우 기반의 클라우드 운용, 프로그램 안정화까지의 현장 실무 경험을 쌓았다.
인턴 기간 중, ‘공장 설비 예지 보전’ 프로젝트에 참여했습니다.
해당 프로젝트는 공장 설비에 진동 센서를 부착하여, 물리적 고장을 사전에 탐지하는 프로젝트로,
센서 네트워크 망 형성 및 DB 구축 및 관리, 데이터 송수신 프로그램 안정화, 환경 구축 및 배포,
그리고 데이터를 실시간으로 시각화하는 대시보드를 구현했다.
